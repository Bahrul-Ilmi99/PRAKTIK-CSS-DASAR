Kesimpulan Skenario 1 : di dalam sebuah container grid,Grid memberikan fleksibilitas lebih dibandingkan dengan metode layout lainnya (seperti Flexbox) karena Anda bisa mengatur posisi elemen dalam grid tertentu menggunakan baris dan kolom, atau bahkan menggabungkan beberapa sel grid.
Kesimpulan Skenario 2 : Dengan mengubah nilai properti display menjadi inline-grid, display: inline-grid; membuat kontainer grid menjadi inline-level, yang berarti ia akan hanya mengambil ruang yang diperlukan untuk kontennya dan dapat berdampingan dengan elemen lain dalam satu baris tanpa memulai baris baru.
Kesimpulan Skenario 3 : Menggunakan properti grid-column-gap dan grid-row-gap dalam elemen grid memungkinkan pengaturan jarak antar kolom dan baris dengan fleksibel. grid-column-gap mengontrol jarak horizontal antar kolom, sedangkan grid-row-gap mengatur jarak vertikal antar baris.
Kesimpulan Skenario 4 : menambahkan properti grid-column-gap dan grid-row-gap ke dalam container dengan display: grid;, tujuan utamanya adalah untuk mengatur jarak antara kolom dan jarak antara baris di dalam grid. Properti-properti ini digunakan untuk memberi ruang di antara elemen-elemen grid dalam baris dan kolom.
